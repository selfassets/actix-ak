Actix-AK API Documentation

AkShare Backend API 文档

基于 Rust Actix-web 实现的期货和股票数据 RESTful API 服务。

基础信息

  • Base URL: https://byteappua-actix-ak.zeabur.app/api/v1
  • 认证方式: Bearer Token
  • 请求头: Authorization: Bearer <token>

目录

  • 健康检查
  • 期货接口
  • 股票接口

快速开始

认证示例

curl -X GET "https://byteappua-actix-ak.zeabur.app/api/v1/health" \
  -H "Authorization: Bearer 12345678"

响应格式

所有接口返回统一的 JSON 格式：

{
  "success": true,
  "data": { ... },
  "error": null
}

错误响应：

{
  "success": false,
  "data": null,
  "error": "错误信息"
}

健康检查

检测服务是否正常运行。

GET /health

健康检查接口，无需认证。

请求示例

curl -X GET "https://byteappua-actix-ak.zeabur.app/api/v1/health"

响应示例

{
  "success": true,
  "data": "服务运行正常",
  "error": null
}

交易所和品种

GET /futures/exchanges

获取支持的交易所列表。

请求示例

curl -X GET "{{baseUrl}}/futures/exchanges" \
  -H "Authorization: Bearer {{token}}"

响应示例

{
  "success": true,
  "data": [
    {
      "code": "SHFE",
      "name": "上期所",
      "description": "Shanghai Futures Exchange"
    },
    {
      "code": "DCE",
      "name": "大商所",
      "description": "Dalian Commodity Exchange"
    }
  ],
  "error": null
}
